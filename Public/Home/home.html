How It Works


Our Process

Here are some of the methods we have been using for our movie and game nights. There are occassions where we don't enforce or follow all of them but in the interest of fairness we try our best to stick to them.

Movie Night:

  • Each person gets one movie nomination, but is not required to nominate
  • We do not have to have a nomination prepared prior to movie night but are generally allowed around 15-20 minutes to find a nomination
  • The person who picked the winning movie is not allowed to nominate during the next movie night
  • After nominations are in we usually take about 5-10 minutes researching the nominations (background, genre, cast etc.)
  • Initial polling is through approval voting, meaning we each vote for any number of the nominations we are interested in watching
  • In the event of a tie, we move to standard runoff voting, where each person votes for one nomination
  • If the runoff vote ends in a tie, we use randomness (coin flip, random number generator etc.)

Game Night:

  • Note: We do not always create polls on Game Night because there are times when we all want to play the same game anyways
  • In the event that there is a wide range of games people want to play, we will create polls
  • There are no limits to the number of game nominations we can add
  • We still use approval voting, but there are no runoffs for ties
  • Since we usually can play multiple games in one Game Night, we will select the top 3 or 4 nominations with the most votes
  • We then will play the games in order of length from shortest game time to longest to ensure every game gets played at least once
  • For example, A game of Codenames or Skribbl.io is much shorter than Rainbow Six Siege or Europa Universalis 4
